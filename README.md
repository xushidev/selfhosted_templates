# Personal Selfhosted Templates

Personal templates registry for selfhosted docker services on [Arcane](https://getarcane.app/).

## Overview

This repository contains Docker Compose template files and setup guides for various self-hosted applications.
Feel free to use them as you wish by adding the registry link to your Arcane instance:

1. Open Arcane Customization
2. Go to Templates
3. Go to Manager Registries
4. Add Registry
5. Enter Registry URL: `https://raw.githubusercontent.com/xushidev/selfhosted_templates/main/registry.json`

## Note for reader

- Make sure to remember to modify the container volume location to be where you want the data to be stored
- Checkout the readme in each folder for some setup guides / documentation (they will often link to the container's official documentation for environment variables)
